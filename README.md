
# 🏦 Agentic AI For Intelligent Account Opening & Onboardibg

An intelligent digital banking onboarding system that automates identity verification using AI agents.

---

## 🚀 Features

- 🔐 Secure User Authentication
- 🧾 PAN Card Verification (AI)
- 🪪 Aadhaar Verification
- 🤳 Face Recognition Matching
- 🚨 Fraud Detection System
- 🧠 Smart Decision Engine (Auto Approve/Reject)
- 📊 Risk Score Visualization (Graphs)
- 📄 PDF Report Generation
- 💾 Auto Save + Resume Application
- 🛠 Admin Dashboard

---

## 🧠 Tech Stack

### Frontend:
- React.js (TypeScript)
- Tailwind CSS
- Framer Motion
- Recharts

### Backend:
- Flask (Python)
- AI Agents

### Database:
- MySQL

---

## 🔄 Workflow

1. User fills account form
2. Upload documents
3. Selfie verification
4. AI agents process:
   - PAN Agent
   - Aadhaar Agent
   - Face Agent
   - Fraud Agent
5. Decision Engine gives:
   - APPROVED / REJECTED / REVIEW
6. Dashboard shows results + graph
